🎨 Images & Media Tags in HTML

1. <img> — Image Tag

Displays an image using the src and alt attributes.

2. <figure> and <figcaption> — Image with Caption

3. <audio> — Audio Tag

Use controls attribute to show play/pause UI.

4. <video> — Video Tag

Use controls, autoplay, loop etc. attributes.

5. <track> — Subtitles / Captions for Video

7. <map> and <area> — Clickable Areas on Image

Define clickable regions inside one image.
