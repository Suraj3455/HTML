HTML Links & Navigation Tags Examples

1. The <a> Tag (Anchor Tag)

The <a> tag defines a hyperlink — used to link to another page, section, or file.

Syntax:

<a href="URL">Link Text</a>

Examples:

Visit Google — basic external link

Open Google in new tab (using target="_blank")

Jump to Navigation Section (Internal Link)

Download PDF File (using download attribute)

Common Attributes:

  • href — destination link or section ID
  • target — where to open link (_blank, _self, _parent, _top)
  • title — tooltip when hovered
  • download — allows downloading file

2. The <nav> Tag (Navigation Section)

The <nav> tag represents a section of navigation links (like menus, headers, or footers).

Example:

<nav>
  <a href="index.html">Home</a>
  <a href="about.html">About</a>
  <a href="services.html">Services</a>
  <a href="contact.html">Contact</a>
</nav>

Rendered Output:

Home About Services Contact

Purpose:

  • Groups together important navigation links.
  • Improves accessibility and SEO (screen readers understand menus).
  • Usually placed in the header or footer.
