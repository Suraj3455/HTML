HTML5 APIs — Tags Included (no JavaScript)

The HTML tags below are the entry points for APIs. Using the APIs requires JavaScript, but the tags themselves are pure HTML:

  • Audio/Video: <audio>, <video>
  • Canvas element (drawing surface): <canvas width="300" height="150"></canvas>
  • SVG inline graphics: <svg> ... </svg>
  • Drag & Drop: attributes like draggable="true" exist on elements
  • Local/Session Storage: storage is provided by the browser via JavaScript APIs (no HTML elements)
  • Geolocation: no HTML element — accessed via JavaScript navigator.geolocation
